The objective of this program is to perform different tasks over given files.

1. The program scans the configuration file from <Config Directory>.
2. After configuration file is processed, the program reads the tasks that should be done based on the given file name and the operation type.
3. Executes the tasks by processing the files that are stored in <ToDo Directory>.
4. The output is registered in files and stored in <doneFiles Directory>.
5. The results of each task are stored in a seperate file name.

The program is in development stage.
At the moment it executes simple tasks like:
- find max value
- fin min value
- get a list of unique words from a given text
- sorting lists
